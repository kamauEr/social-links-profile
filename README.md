CSS - How to set different button states using the LVHA (link,visited,hover,active) pseudo states in order.

```css
.button li a:link,
.button li a:visited {
  background-color: hsl(0, 0%, 20%);
  color: hsl(0, 0%, 100%);
}

.button li a:hover,
.button li a:active {
  background-color: hsl(75, 94%, 57%);
  color: hsl(0, 0%, 8%);
}
```
